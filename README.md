# dotfiles

## prerequirement

- git
- vim 8.*
- (npm, nodejs)

## installed tools

- dein
- tig

## install

```
$ cd $HOME
$ git clone https://github.com/yCroma/dotfiles.git
$ cd dotfiles
$ ./install
$ ./symblik_link
```

first: use call dein function in vim

```
:call dein#recache_runtimepath()
:call dein#install()
```
